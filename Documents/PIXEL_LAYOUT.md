# Pixel Layout Notes

Reference resolution: `1920x1080`

Base grid:
- `12px`

Grid coverage at `1920x1080`:
- Play field: `160x90` cells

Element sizes:
- Player: `48x48 px` = `4x4` grid cells
- Flag: `24x120 px` = `2x10` grid cells
- Circle: `120x120 px` = `10x10` grid cells

Notes:
- `12px` grid measures out cleanly and fits the current visual targets well.
- Other element sizes also align cleanly to this grid.
