# Sling Hero Color Palette

Палітра сформована на основі наданого референсу екрана вибору рівнів.

## Base / Background

- `#0F0B1C` — майже чорний фіолетовий
- `#1B152A` — глибокий фон
- `#2A2238` — темний UI / тіні
- `#3D334B` — середній фіолетово-сірий
- `#5A4E69` — камінь / неактивні плитки

## Primary Accent

- `#C93A52` — основний червоно-рожевий
- `#E05263` — яскравий акцент
- `#8E2338` — темний червоний для тіней
- `#F06B7A` — світлий highlight для кнопок / рамок

## Secondary Accent

- `#D67A2D` — помаранчевий glow
- `#F2B24A` — золотий акцент
- `#8D5A2A` — темний бронзовий

## Text / Light

- `#F5EEDF` — теплий майже білий
- `#D8CFC3` — приглушений світлий
- `#A79BAF` — вторинний текст / неактивний UI

## Nature Accent

- `#556B2F` — оливковий зелений
- `#7B8F3B` — світліша трава
